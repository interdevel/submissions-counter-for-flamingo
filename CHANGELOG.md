## [1.0.0] - 2024-01-01

### Added

- Initial release of the Submissions Counter for Flamingo plugin.
- `[flamingo_count]` shortcode to display total submissions stored by Flamingo.
- `form_title` attribute to filter by a specific form name.
- `html_tag` attribute to wrap the count in an HTML element with class `fsc-count`.
